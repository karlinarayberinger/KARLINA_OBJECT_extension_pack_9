The following hyperlinks each lead to exactly one raw file which is part of the public GitHub repository named RAW_GITHUB_FILES_MACRO_DIRECTORY_first_75_micro_directories. The purpose of this web page is to save karbytes time and energy when saving each raw file in that GitHub repository to the WayBack Machine (as a web page which can be publicly viewed on the WayBack Machine database). karbytes intends to routinely log into its Internet Archive user account in order to access the batch save feature of the WayBack Machine (when karbytes saves the Uniform Resource Locator) of this web page).

Note that, for a directory web page which has more than a specific number of hyperlinks on it (i.e. “many” hyperlinks), some of (but not all of) the web pages which are referenced via hyperlinks on that directory web page are saved in a “random” order during a batch WayBack Machine save operation (and a batch save occurs as a result of selecting the “save outlinks” option on the WayBack Machine “Save Page Now” input form). A directory web page with “many” hyperlinks on it will need to be saved multiple times in order for each web page which is referenced by a hyperlink on that directory web page to be saved via batch save operation.

Note that each of the source code files which are referenced via hyperlink in the list below are not the most up-to-date version of its corresponding web page. The most up-to-date version of each of those source code files is available in the directory web page named raw_github_files_directory_KARLINA_OBJECT_extension_pack_9.
